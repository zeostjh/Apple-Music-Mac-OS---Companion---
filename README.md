# companion-module-apple-music

Bitfocus Companion module to control the macOS Music app (Apple Music).

## Tested On 
- Companion Version 4.2.5
- Mac OS 26.2 

## Features
- Full transport control: Play, Pause, Toggle, Stop, Next, Previous
- Live variables for track name, artist, album, and time (elapsed / remaining / total)
- Feedbacks for playing/paused state
- Included presets for quick setup

## Installation (Development)
1. Clone or download this folder
2. Place it inside your Companion **Developer modules path**
3. Run `yarn install` inside the folder
4. Restart or reload the module in Companion

## macOS Permissions
You may need to allow Automation access for Music in:
**System Settings → Privacy & Security → Automation**
