# Apple Music

This module controls the Apple Music app on macOS through AppleScript.

Available actions:

- Play / pause toggle
- Play
- Pause
- Next track
- Previous track
- Refresh metadata

Available variables:

- `$(applemusic:track_name)`
- `$(applemusic:artist_name)`
- `$(applemusic:album_name)`
- `$(applemusic:player_state)`
